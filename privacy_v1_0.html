1. Общие положения

Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с Федеральным законом Российской Федерации от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки и защиты персональных данных пользователей мобильного приложения AutoTrat (далее — «Приложение»).

Используя Приложение, Пользователь подтверждает согласие с настоящей Политикой и условиями обработки персональных данных.

2. Персональные данные Пользователей

Оператор может обрабатывать следующие категории персональных данных Пользователя:

2.1. Данные, вводимые Пользователем самостоятельно

  • марка и модель автомобиля;
  • наименование автомобиля (опционально);
  • цвет автомобиля;
  • объём двигателя;
  • тип коробки передач;
  • год выпуска;
  • тип топлива;
  • начальный и текущий пробег;
  • сумма расходов;
  • количество заправленного топлива (литры);
  • дата внесения расходов;
  • комментарий (опционально);
  • фото чеков (опционально);
  • пользовательские категории расходов.

2.2. Технические и служебные данные

  • данные, необходимые для отправки push-уведомлений;
  • данные, связанные с авторизацией в Яндекс Диске для резервного копирования;
  • данные о резервных копиях (бэкапах).

3. Цели обработки персональных данных

Персональные данные обрабатываются в следующих целях:

  • предоставление функциональности Приложения;
  • ведение учёта расходов на автомобиль;
  • формирование статистики на основе введённых Пользователем данных;
  • экспорт данных в форматы PDF, CSV, Excel;
  • отправка push-уведомлений (о страховке, бюджете и иных событиях);
  • создание резервных копий данных по инициативе Пользователя;
  • предоставление премиум-функций Приложения.

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании добровольного согласия Пользователя. Согласие предоставляется Пользователем посредством подтверждения в модальном окне перед началом использования Приложения.

5. Условия хранения и передачи данных

Персональные данные Пользователя хранятся локально на устройстве Пользователя. По желанию Пользователя может осуществляться резервное копирование данных на его личный аккаунт в сервисе Яндекс Диск.

  • Передача данных осуществляется только сервису Яндекс Диск и ограничена доступом к одной скрытой папке.
  • Все данные, передаваемые на Яндекс Диск, шифруются.
  • Оператор не имеет доступа к содержимому пользовательских резервных копий.
  • Пользователь вправе удалить резервную копию с Яндекс Диска.

6. Защита персональных данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения, включая использование шифрования.

7. Права Пользователя

Пользователь имеет право:

  • получать информацию об обработке своих персональных данных;
  • требовать уточнения, блокирования или уничтожения персональных данных;
  • отозвать согласие на обработку персональных данных.

8. Отзыв согласия

Пользователь может отозвать согласие на обработку персональных данных:

  • через соответствующую кнопку в Приложении;
  • путём направления письма на электронную почту: AutoTart@yandex.com.

Отзыв согласия может повлечь невозможность дальнейшего использования функциональности Приложения.

9. Обработка данных несовершеннолетних

Приложение предназначено для лиц старше 18 лет. Оператор не осуществляет целенаправленную обработку персональных данных несовершеннолетних.

10. Автоматизированная обработка

В Приложении не осуществляется автоматизированное принятие решений, порождающих юридические последствия для Пользователя.

11. Изменение Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия Политики доступна Пользователю в любое время.

12. Контактная информация

По всем вопросам, связанным с обработкой персональных данных, Пользователь может обратиться по адресу: AutoTart@yandex.com.
