1. Общие положения

Настоящая Политика разработана в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных» и определяет порядок обработки и защиты персональных данных пользователей мобильного приложения AutoTrat.

Используя Приложение, Пользователь выражает согласие с настоящей Политикой.

2. Категории обрабатываемых данных

2.1. Данные, вводимые Пользователем

  • данные об автомобиле (марка, модель, год, тип топлива и др.);
  • данные о расходах (суммы, даты, категории, комментарии);
  • пробег автомобиля;
  • пользовательские категории расходов;
  • фотографии чеков и автомобиля (опционально).

2.2. Технические и служебные данные

  • данные, необходимые для работы push-уведомлений;
  • технические данные, связанные с резервным копированием;
  • данные, связанные с авторизацией через Яндекс ID (OAuth).

Приложение не собирает данные о геолокации, контактах, звонках, содержимом сообщений и иных данных, не относящихся к функциональности Приложения.

3. Цели обработки персональных данных

  • предоставление функциональности Приложения;
  • учёт и анализ расходов на автомобиль;
  • отображение статистики и отчётов;
  • экспорт данных в PDF, CSV и Excel;
  • отправка push-уведомлений;
  • создание резервных копий по инициативе Пользователя;
  • предоставление премиум-функций.

4. Правовые основания обработки

Обработка осуществляется на основании добровольного согласия Пользователя, выраженного при первом запуске Приложения.

5. Хранение и передача данных

Все персональные данные хранятся локально на устройстве Пользователя.

5.1. Резервное копирование

  • Резервное копирование осуществляется только по инициативе Пользователя.
  • Бэкапы сохраняются в личный аккаунт Пользователя в сервисе Яндекс Диск.
  • Данные шифруются перед передачей.
  • Оператор не имеет доступа к содержимому резервных копий.
  • Пользователь вправе удалить резервную копию в любое время.

5.2. Системный Android-бэкап

Операционная система Android может автоматически сохранять часть настроек приложения (например, пользовательские параметры). Финансовые данные и база расходов в системный бэкап не включаются.

6. Виджеты и системные интерфейсы

Приложение поддерживает системные виджеты Android, которые позволяют добавлять расходы и просматривать статистику. Виджеты используют только локальные данные и не передают информацию третьим лицам.

7. Push-уведомления

Приложение может отправлять push-уведомления, связанные с напоминаниями, бюджетами и событиями, указанными Пользователем. Push-уведомления не содержат чувствительных персональных данных.

8. Защита персональных данных

Оператор применяет необходимые технические и организационные меры защиты, включая шифрование и ограничение доступа к данным.

9. Права Пользователя

  • получать информацию об обработке своих данных;
  • удалять свои данные;
  • отзывать согласие на обработку.

10. Отзыв согласия и удаление данных

Пользователь может отозвать согласие через интерфейс Приложения либо направив запрос на электронную почту Оператора.

При отзыве согласия все локальные данные удаляются. Резервные копии на Яндекс Диске удаляются по выбору Пользователя.

11. Обработка данных несовершеннолетних

Приложение предназначено для лиц старше 18 лет.

12. Изменение Политики

Актуальная версия Политики всегда доступна в Приложении.

13. Контакты
