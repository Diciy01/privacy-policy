1. Общие положения

Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с Федеральным законом Российской Федерации от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки и защиты персональных данных пользователей мобильного приложения AutoTrat (далее — «Приложение»).

Используя Приложение, Пользователь подтверждает, что он ознакомлен с настоящей Политикой и даёт согласие на обработку своих персональных данных в объёме, целях и на условиях, изложенных в настоящей Политике.

Обработка персональных данных осуществляется на принципах законности, справедливости, минимальности, целевого использования и обеспечения безопасности персональных данных.

2. Категории обрабатываемых персональных данных

2.1. Данные, вводимые Пользователем

  • данные об автомобиле (марка, модель, год выпуска, тип топлива и иные характеристики);
  • данные о расходах (суммы, даты, категории, комментарии);
  • данные о пробеге автомобиля;
  • пользовательские категории расходов;
  • фотографии чеков и автомобиля (опционально).

2.2. Технические и служебные данные

  • данные, необходимые для работы push-уведомлений;
  • технические данные, связанные с резервным копированием;
  • данные, связанные с авторизацией через сервис Яндекс ID (OAuth).

Приложение не осуществляет сбор данных о геолокации, контактах, телефонных вызовах, содержимом сообщений, а также иных данных, не относящихся к функциональности Приложения.

3. Цели обработки персональных данных

Персональные данные обрабатываются исключительно в следующих целях:

  • предоставление функциональности Приложения;
  • учёт и анализ расходов на автомобиль;
  • формирование статистики и отчётов;
  • экспорт данных в форматы PDF, CSV и Excel;
  • отправка push-уведомлений;
  • создание резервных копий по инициативе Пользователя;
  • предоставление премиум-функций Приложения.

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании добровольного согласия Пользователя, выраженного при первом запуске Приложения и подтверждённого дальнейшим использованием Приложения.

5. Хранение и передача персональных данных

Все персональные данные Пользователя хранятся локально на устройстве Пользователя.

5.1. Резервное копирование

  • резервное копирование осуществляется исключительно по инициативе Пользователя;
  • резервные копии сохраняются в личный аккаунт Пользователя в сервисе Яндекс Диск;
  • данные шифруются перед передачей;
  • Оператор не имеет доступа к содержимому резервных копий;
  • Пользователь вправе удалить резервную копию в любое время самостоятельно.

5.2. Системный Android-бэкап

Операционная система Android может автоматически сохранять часть пользовательских настроек приложения (например, параметры интерфейса). Финансовые данные и база расходов не включаются в системный бэкап Android.

5.3. Платёжная информация

Оплата премиум-функций Приложения осуществляется с использованием платёжной инфраструктуры RuStore. Платёжные данные Пользователя (включая данные банковских карт и иные платёжные реквизиты) обрабатываются непосредственно платёжной системой RuStore и не передаются Оператору.

Оператор получает и обрабатывает только информацию о статусе подписки (активна / неактивна) в целях предоставления доступа к премиум-функциям Приложения.

6. Виджеты и системные интерфейсы

Приложение поддерживает системные виджеты Android, позволяющие добавлять расходы и просматривать статистику. Виджеты используют только локальные данные и не передают информацию третьим лицам.

7. Push-уведомления

Приложение может отправлять push-уведомления, связанные с напоминаниями, бюджетами и событиями, указанными Пользователем. Push-уведомления не содержат чувствительных персональных данных.

8. Защита персональных данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения, включая:

  • использование шифрования;
  • ограничение доступа к данным;
  • использование защищённых механизмов хранения.

9. Права Пользователя

Пользователь имеет право:

  • получать информацию об обработке своих персональных данных;
  • требовать удаления своих персональных данных;
  • отозвать согласие на обработку персональных данных.

10. Отзыв согласия и удаление данных

Пользователь может отозвать согласие на обработку персональных данных через интерфейс Приложения либо направив запрос на электронную почту Оператора.

При отзыве согласия:

  • локальные данные удаляются с устройства Пользователя;
  • резервные копии на Яндекс Диске удаляются по выбору Пользователя.

11. Обработка данных несовершеннолетних

Приложение предназначено для лиц старше 18 лет. Оператор не осуществляет целенаправленную обработку персональных данных несовершеннолетних.

12. Изменение Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия Политики всегда доступна Пользователю в Приложении.
